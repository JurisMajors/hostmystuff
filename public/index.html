Welcome to HostMyStuff!

HostMyStuff is a private paste bin which supports syntax highlighting (with automatic language detection) and server side file management

See the source code and more information here.

Authentication

To use this service you must have an api-key

How can I get one?

Contribute to the project or ask me for one.

Request examples

Uploading: 


curl --header "key: api-key" -F"file=@fileName" https://hostmystuff.xyz/

Deleting: 


curl --header "key: api-key" -X DELETE https://hostmystuff.xyz/fileName

Listing files attached to your key: 


curl --header "key: api-key" https://hostmystuff.xyz/allfiles

Listing all info about you: 


curl --header "key: api-key" https://hostmystuff.xyz/allinfo

Syntax highlighting

HostMyStuff supports syntax highlighting in many styles and for many languages. See this hello world programme with some highlight sugar



public class HelloWorld {
	public static void main(String... args) {
		System.out.println("Hello World!");	
	}	
}

To not apply syntax highlighting simply supply a raw query parameter to the GET request ?raw=true.

Useful in-case of bugs when highlighting is applied (please report them if you find any) or in case you want to download the file (e.g. wget).

Other information

Max file size is 512M

Files get deleted according to the same formula as 0x0

retention = min_age + (-max_age + min_age) * pow((file_size / max_size - 1), 3)
